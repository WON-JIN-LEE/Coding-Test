function solution(n) {
  var answer = 0;
  if (n === Math.pow(Math.floor(Math.sqrt(n)), 2)) {
    answer = Math.pow(Math.sqrt(n) + 1, 2);
    return answer;

## Math.sqrt()

- Math.sqrt() 함수는 숫자의 제곱근을 반환합니다.

```javascript
Math.sqrt(9); // 3
Math.sqrt(2); // 1.414213562373095
Math.sqrt(1); // 1
Math.sqrt(0); // 0
Math.sqrt(-1); // NaN
```

## Math.floor()

- Math.floor() 함수는 주어진 숫자와 같거나 작은 정수 중에서 가장 큰 수를 반환합니다.

```javascript
Math.floor(45.95); //  45
Math.floor(45.05); //  45
Math.floor(4); //   4
Math.floor(-45.05); // -46
Math.floor(-45.95); // -46
```

## Math.pow()

- Math.pow()함수는baseexponent처럼 base 에 exponent를 제곱한 값을 반환합니다.

```javascript
// 간단한 예
Math.pow(7, 2); // 49
Math.pow(7, 3); // 343
Math.pow(2, 10); // 1024
// 분수 지수
Math.pow(4, 0.5); // 2 (4의 제곱근)
Math.pow(8, 1 / 3); // 2 (8의 세제곱근)
```
